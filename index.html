LlamaCSS Framework Guide

This is a guide for the LlamaCSS Framework. This guide is made by BurnyLlama who also made framework.
This guide page is of course built on the framework and will go over most of the stuff.

Is there any advantage using a css-framework?
Yes - if you use a css-framework and update the page there will be no design issues, as you may have encountered when making pages and making changes to your css your browser sometimes caches the old styles and can only display the old styles. Having a css-framework will fix this.
Also devoloping a new page will be faster because you don't have to make a new stylesheet.


To use the framework easily add this in your HTML head-tag:

<link href="https://burnyllama.tk/css/main.css" rel=“stylesheet”>

Text-based features

Basic text

Using text is mostly simple. You get normal text with paragraphs.
You can also get bold text with the b-tag, italic with the i-tag, both with the em-tag. Lastly you can use the u-tag for underlining.

italic
emphasised

Important/Note box

You can use the "note" class on nearly any element to create one of these boxes.
I would recommend a paragraph though.

As of now you can only change color of the text and background. I will probably work on classes that will make you able to change the color of the border.

Color features

I've made som color-classes that comes preloaded with the framework:

  • red
  • yellow
  • green
  • blue
  • cyan
  • purple
  • black
  • white
  • transparent

By adding a "BG" in the end of the class you give the background another color:

  • red
  • yellow
  • green
  • blue
  • cyan
  • purple
  • black
  • white
  • transparent

By adding a "D" or a "B" for a Darker or Brighter color.
Important! If you want to do the same with the background add the "B" or "D" before the "BG".

You can go to /palette/colors.css and customize the colors.
Even add new colors although not really that recommended.
